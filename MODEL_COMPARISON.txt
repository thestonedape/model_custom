SECTION 5 — KEY DIFFERENCES EXPLAINED
================================================================================

1. WHY REPLICA WORKED BUT ENHANCED PLATEAU'D INITIALLY
   ─────────────────────────────────────────────────────
   Replica:   VQ loss active (lambda=1.0) → codebook trains alongside encoder
              Encoder outputs and codebook vectors co-adapt from random init
              VQ serves as a meaningful learned intermediate representation

   Enhanced:  VQ loss disabled (lambda=0.0) to prevent divergence (see below)
              BUT model still routed through VQ layer → frozen random codebook
              Every encoder output snapped to same random bucket every step
              Classifier learned fixed random_bucket → word mapping
              Encoder had zero incentive to change (VQ buckets never moved)

2. WHY VQ LOSS DIVERGED IN ENHANCED (lambda=1.0 attempt)
   ───────────────────────────────────────────────────────
   VQ loss = MSE(encoder_output, nearest_codebook_vector)
   At init: encoder outputs ~N(0,1), codebook ~N(0, small)
   As CE loss trains encoder, output magnitudes grow (4–16 range)
   Codebook never catches up → MSE explodes (0.4 → 240+ over 100 batches)
   With lambda=1.0: total_loss = CE(~6) + 1.0 * VQ(240+)
   Gradient clipping suppresses everything → model cannot learn CE

   Root cause: VQ codebook needs to be initialized in the same scale as the
   encoder outputs, or needs a warm-up period before being active.
   Replica avoided this because its encoder was pre-at a good scale from SGD.

3. OPTIMIZER DIFFERENCE: SGD vs AdamW
   ─────────────────────────────────────
   SGD (replica): Simple, stable, well-understood for this type of model
                  lr=1e-4 conservative → slow but steady convergence
   AdamW (enhanced): Adaptive per-parameter LR, faster initial convergence
                     But with lr=5e-4 may overshoot on this small signal EEG task
                     Weight decay decoupled (better generalization theory)

4. CONTRASTIVE LOSS: DISABLED IN BOTH
   ─────────────────────────────────────
   Ablation study in replica showed alpha=0.1-0.9 ALL blocked CE learning
   Both models use alpha=0.0 (contrastive loss completely disabled)
   See ANALYSIS_RESULTS section in ENHANCED_MODEL_ANALYSIS.txt for full ablation
